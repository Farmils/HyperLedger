(inner) buildCCPOrg(organization) → {Object}

Метод buildCCPOrg в Hyperledger Fabric используется для построения и возврата объекта Connection Profile (CCP) для указанной организации. Connection Profile — это JSON-файл, который содержит информацию о сети Hyperledger Fabric, включая сведения о пирах (peers), сертификатах, каналах и других компонентах сети. Этот файл необходим для настройки клиентских приложений, которые взаимодействуют с сетью Hyperledger Fabric.
